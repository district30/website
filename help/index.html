District30- OctoBoot
Manage your website with octoboot



Pour une efficacité optimale de la plateforme, il est conseillé d'utiliser le navigateur Chrome

CONNEXION A LA PLATEFORME

  1. Aller à l'adresse octoboot.ovh. La connexion se fait via un compte github. Il suffit de suivre les instructions et de se connecter.

  2. Choisir le projet (i.e le site) à modifier.

UTILISATION GENERALE : Modifier du texte

  1. Dans la barre d'outils à gauche, cliquer sur le mode édition 'Edit'.

  2. SOIT il est possible de modifier directement le texte en double-cliquant sur celui-ci

  3. SOIT il est également possible de modifier en survolant le texte avec la souris et en appuyant sur la touche espace du clavier.

  4. SAUVEGARDER dans la barre d'outils en cliquant sur le bouton 'Save'.

Voir la vidéo

UTILISATION GENERALE: Modifier une image

  1. Préparer le(s) fichier(s) image à ajouter (ou modifier). Attention au nom des fichiers, il est préférable d'éviter les espaces, les apostrophes, les caractères spéciaux,... 

  2. Uploader à l'aide du bouton 'Upload' de la barre d'outils, le(s) fichiers dans la bibliothèque du backoffice.

  3. Dans la barre d'outils, passer en mode édition 'Images'. Au clic , une barre regroupant toutes les images apparaît. Soit il y a un champs de recherche soit les images sont rangées par ordre d'apparition, il faut donc scroller. Cliquer sur l'image, celle-ci blink (clignote) quand elle est visible. Double cliquer et modifier l'image.
    Remarque :  le champs de texte alternatif sert au référencement Google.

  4. SAUVEGARDER

Voir la vidéo

UTILISATION GENERALE: Ajouter des photos dans une galerie

  1.  Préparer les fichiers: 2 pour chaque photo. La petite vignette carré doit être au format 200x200 px et la grande doit respecter le ratio xxxX1000px (1000px en hauteur)

  2. Uploader les fichiers à l'aide du bouton 'Upload' de la barre d'outils.

  3. Dans la barre d'outils cliquer sur 'Duplicate' et dupliquer le nombre de photos à ajouter.

  4. Dans la barre d'outils, passer en mode édition 'Images'. Au clic , une barre regroupant toutes les images apparaît. Soit il y a un champs de recherche soit les images sont rangées par ordre d'apparition, il faut donc scroller. Cliquer sur l'image, celle-ci blink (clignote). Double cliquer et modifier l'image. Ne pas oublier de changer la petite et la grande
    Remarque :  le champs de texte alternatif sert au référencement Google.

  5. SAUVEGARDER

Voir la vidéo

UTILISATION GENERALE : Insérer un lien sur un bouton

  1. Passer en mode édition 'edit' sur la barre d'outils. Cliquer sur le bouton sur lequel ajouter un lien. Dans la barre de sélection, cliquer sur le logo représentant 2 maillons 'link'

  2. Il existe 2 sortes de liens:
    - Les liens internes au site. Paramétrée par défaut, la fenêtre de défilement des différentes pages du site s'ouvre. Sélectionner la page ou le fichier de destination. Celle-ci peut soit s'ouvrir dans le même onglet (fenêtre) que la précédente --> option 'self', soit dans un nouveau --> option blank. Pour les sites extérieur à District (youtube par exemple) il est recommandé de choisir l'option 'blank'.
    - Les liens externes au site. Pour y accéder il suffit de cliquer en dehors du menu déroulant qui s'ouvre par défaut à l'ajout d'un lien. Dans la fenêtre http il suffit de coller le lien. Valider. Par convention et pour le confort de l'internaute, les liens externes s'ouvrent dans un nouvel onglet --> option 'blank'.
     

  3. Quitter le mode édition 'edit'

  4. SAUVEGARDER

UTILISATION GENERALE : Modifier la couleur d'un titre

  1. Dans la barre d'outils à gauche, cliquer sur le mode édition 'Edit'.

  2. Cliquer sur le texte à modifier, dans la barre de sélection cliquer sur le crayon 'Style CSS'. Une nouvelle fenêtre en bas s'ouvre avec toutes les fonctions CSS possibles.

  3. Dans la barre de recherche, chercher color et modifier soit avec les références RGB soit avec la palette.

  4. SAUVEGARDER dans la barre d'outils en cliquant sur le bouton 'Save'.

Voir la vidéo

UTILISATION GENERALE : Modifier la couleur de fond d'une section

  1. Dans la barre d'outils à gauche, cliquer sur le mode édition 'Edit'.

  2. Il s'agit de sélectionner la section. Sélecionner n'importe quel 'bloc' dans la section et remonter dans les blocs en utilisant la flèche haue de la barre de sélection. Le nom du bloc apparaît en haut à droite. Il suffit de s'arrêter une fois que dans la barre de sélection apparait 'SECTION'. 

  3. Dans la barre de sélection cliquer sur le crayon 'Style CSS'. Une nouvelle fenêtre en bas s'ouvre avec toutes les fonctions CSS possibles.

  4. Dans la barre de recherche, chercher 'background color' et modifier soit avec les références RGB soit avec la palette.

  5. SAUVEGARDER dans la barre d'outils en cliquant sur le bouton 'Save'.

Voir la vidéo

UTILISATION GENERALE: Supprimer un fichier image uploadé dans la bibliothèque

  1.  Cliquer sur la barre de navigation interne (/website/....) comme pour changer de page.

  2. Dans le menu déroulant qui s'ouvre, les images uploadées apparaissent également, /website/uploads/NOMDELIMAGE.png par exemple. Sélectionner le bon fichier. Il est possible d'utiliser uniquement le nom du fichier pour le retrouver.

  3. Le fichier image s'ouvre

  4. Dans la barre d'outils à gauche, cliquer sur 'Delete'

  5. SAUVEGARDER

UTILISATION GENERALE: OUPS j'ai tout cassé !

  1. Il arrive qu'après une sauvegarde on se rende compte qu'on a touché à la structure du site et qu'on a "cassé" la page. Octoboot permet donc de revenir à des sauvegardes antérieures pour revenir à une version fonctionelle.

  2. Dans la barre d'outils cliquer sur 'Save' puis sur 'Restore'

  3. Une nouvelle barre d'outils apparaît reprenant toutes les dernières sauvegardes (différenciée par la date et l'heure).

  4. Cliquer sur la version qui vous intéresse pour la charger. Si c'est bien la bonne, dans la nouvelle barre d'outils, cliquer sur 'Save' pour que cette version redevienne celle par défaut sinon cliquer sur 'Cancel' pour revenir à la dernière sauvegarde.

  5. SAUVEGARDER

PUBLIER VOTRE SITE SUR UNE PAGE DE TEST

  1. Avant une mise en ligne effective du site, il convient de tester les modifications sur une page de test.

  2. Dans la barre d'adresse interne à Octoboot cliquer en haute à droite sur 'Preview'. Une nouvelle fenêtre de la page s'ouvre

  3. Vérifier les modifications

Voir la vidéo

MISE EN LIGNE DU SITE

  1. Une fois les modifications vérifiées. Vous pouvez procéder à la mise en ligne effective du site

  2. Dans la barre d'outils, cliquer sur 'Publish', puis dans la partie Github sur 'Publish'.
    Remarque: la mise à jour effective du site peut prendre plusieurs minutes. Avant de vérifier si la nouvelle version du site est bien en ligne, vider votre cache navigateur
